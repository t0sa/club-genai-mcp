# Agent Rules

## After every code change

1. **Build** — Rebuild the project and verify it compiles without errors before proceeding.

2. **Write tests** — Write or update tests that cover the modified code.

3. **Run tests** — Execute the full test suite and confirm all tests pass with no regressions.

4. **Lint** — Run the appropriate linter for the language/framework in use and fix any reported issues.
