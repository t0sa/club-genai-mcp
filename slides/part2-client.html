Pratique : MCP GitHub

Configuration & Auth

Ajouter le serveur mcp-github et générer un token d'accès depuis Github.

Configuration à ajouter dans mcp.json 
    {
        "mcpServers": {
            "github": {
            "url": "https://api.githubcopilot.com/mcp/",
            "headers": {
                "Authorization": "Bearer votre_token"
            }
            }
        }
    }
