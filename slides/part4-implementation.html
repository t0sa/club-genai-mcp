Phase 1 — Serveur "Hello World"

OBJECTIF

Créer un serveur TypeScript qui expose un outil add_numbers basique.

TESTER AVEC L'INSPECTEUR

npx @modelcontextprotocol
  /inspector

Interface visuelle pour envoyer des requêtes et voir les réponses en temps réel.

CODE COMPLET DU SERVEUR

import { McpServer } from "@modelcontextprotocol/sdk";
import { StdioServerTransport } from "@modelcontextprotocol/sdk";
import { z } from "zod";

const server = new McpServer({
  name: "hello-mcp", version: "1.0.0"
});

server.tool("add_numbers",
  { a: z.number(), b: z.number() },
  async ({ a, b }) => ({
    content: [{ type: "text", text: String(a + b) }]
  })
);

await server.connect(new StdioServerTransport());

Phase 2 — Configuration de l'Hôte

Connecter ton serveur à Claude Desktop pour que l'IA puisse l'utiliser.

FICHIER DE CONFIG

// ~/Library/Application Support/
// Claude/claude_desktop_config.json
{
  "mcpServers": {
    "hello-mcp": {
      "command": "node",
      "args": ["./dist/index.js"]
    }
  }
}

VÉRIFICATION

1 Redémarrer Claude Desktop
2 Chercher l'icône 🔌 dans le chat
3 Tester : "Calcule 5 + 3 avec mon outil"
✓ Claude appelle add_numbers et retourne 8 !

L'IA choisit automatiquement l'outil approprié selon le contexte de la conversation.

Phase 3 — Client Personnalisé

Développer son propre script client — sans dépendre d'une interface tierce.

CODE DU CLIENT

import { Client } from "@modelcontextprotocol/sdk";
import { StdioClientTransport } from "@modelcontextprotocol/sdk";

const transport = new StdioClientTransport({
  command: "node",
  args: ["./dist/index.js"]
});

const client = new Client(
  { name: "my-client", version: "1.0.0" },
  { capabilities: {} }
);

await client.connect(transport);
const tools = await client.listTools();
const result = await client.callTool(
  "add_numbers", { a: 5, b: 3 }
);

Pourquoi un client custom ?

  • 🤖 Automatiser des workflows complets
  • 🔗 Chaîner plusieurs appels d'outils
  • 🧩 Intégrer dans une app existante
  • 🛡️ Contrôler finement les permissions

Tu deviens indépendant de Claude Desktop — ton outil peut être utilisé dans n'importe quel contexte !
