30 MINUTES

Nouveau TP : Interface Front-End

Créez une interface visuelle pour afficher les tâches ! Déléguez la construction du front à votre IA en utilisant Nuxt et Playwright via leurs serveurs MCP officiels.

Étape 1 : Le Front par l'IA

Utiliser Nuxt MCP pour générer le code du front, et Playwright MCP pour tester visuellement le rendu en direct.

Étape 2 : Connecter l'API

Récupérez vos données via le serveur "Task Tracker" que vous venez de coder. Connectez-y le front pour afficher les messages.

Installation des MCP Externes

Dans Claude Code

Exécutez dans votre terminal :

claude mcp add nuxt npx --yes @nuxt/mcp
claude mcp add playwright npx --yes @playwright/mcp-server
Dans VS Code (GitHub Copilot)

Éditez les paramètres MCP JSON et ajoutez :

  • nuxt 
    npx --yes @nuxt/mcp
  • playwright 
    npx --yes @playwright/mcp-server

Consigne Complète du TP

1️⃣
Le Scaffold Initial

Démarrez votre projet Nuxt. Demandez à l'IA de créer le design via les outils du Nuxt MCP et de le tester visuellement via Playwright MCP.

2️⃣
Le Pont de Données (GET)

Intégrez les données du "Task Tracker". Demandez aux LLMs de lire votre serveur local via leur architecture MCP pour populer la page Nuxt.

⚡
Interactions Avancées (POST)

Ajoutez un filtre ou l'envoi de tâches en implémentant un client MCP dans le front/back Nuxt qui communiquera avec votre propre serveur.
