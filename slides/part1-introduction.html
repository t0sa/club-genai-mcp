Qu'est-ce que le MCP ?

L'Analogie Parfaite : MCP est à l'IA ce que le port USB-C est à l'électronique. Un standard universel pour connecter les IA à des outils externes.

Les Acteurs
  • 🖥️ Host : App globale (Claude Desktop, VS Code)
  • 🔗 Client : Composant de connexion
  • ⚙️ Server : App exposant les outils
Les Couches
  • 🗂️ Data Layer : JSON-RPC 2.0 (cycle de vie)
  • 📡 Transport Layer : Stdio (local) ou HTTP/SSE (distant)

Architecture : Host - Client - Serveur

💡

Dans un IDE (VS Code, Cursor) ou un terminal (Claude Code), le Host et le Client sont intégrés dans le même processus — c'est l'application elle-même qui gère la connexion.
